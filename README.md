# Industrial Core - Odoo Enterprise Module

## Overview
Custom Odoo 17 module developed for production/manufacturing enterprises in Addis Ababa. 

## Features
* **Waste Tracking**: Automatically computes loss percentages during manufacturing.
* **Automated Invoicing**: Directly links production outputs to the Accounting Dashboard.
* **Security**: Role-based access for Managers vs. Factory Workers.

## Technical Stack
* **Core**: Odoo 17
* **Database**: PostgreSQL
* **Logic**: Python / XML (XPath)
